# Sample Project for saikumarbobbala

This is a small sample Node.js project to demonstrate a minimal repository structure.

Quick start:
1. Clone the repo and checkout the branch:
   ```
   git checkout sample-project
   ```
2. Install (if needed):
   ```
   npm install
   ```
   (This project has no external dependencies.)
3. Run:
   ```
   node index.js
   ```
4. Open http://localhost:3000 in your browser — you should see "Hello from sample project!".

Files included:
- index.js — minimal HTTP server
- package.json — project metadata
- .gitignore — ignore node_modules
- LICENSE — MIT
- README.md — this file
